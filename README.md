# Aurora Theater Web Application

Welcome to the **Aurora Theater** project! This web application is designed to provide a seamless and inspiring online experience for theater visitors, staff, and administrators. With features for ticket management, barcode scanning, and event information, Aurora Theater brings the magic of live performances to your fingertips.

---

## 🚀 Features

- **Home Page:** Welcoming introduction and quick navigation to all main sections.
- **Voorstellingen:** Browse and discover upcoming shows and performances.
- **Medewerkers:** Meet the passionate team behind every successful event.
- **Tickets:** Reserve, view, and scan tickets with real-time barcode validation.
- **Barcode Scanning:** Integrated camera-based barcode scanning for validating ticket authenticity at the entrance.
- **Responsive Design:** Optimized for desktop and mobile devices.
- **Modern UI:** Built with Bootstrap 5 and custom styling for a professional look.

---

## 🛠️ Technologies Used

- **PHP** (MVC structure)
- **MySQL** (database)
- **Bootstrap 5** (frontend framework)
- **JavaScript** (barcode scanning with [Html5-QRCode](https://github.com/mebjas/html5-qrcode))
- **Font Awesome** (icons)

---

## 📦 Installation

1. **Clone the repository:**
    ```bash
    git clone https://github.com/omid2831/theater.git
    ```
2. **Import the database:**
    - Use the `app/db/createscript.sql` to create all necessary tables.
3. **Configure your environment:**
    - Update `app/config/config.php` with your database credentials and URLROOT.
4. **Run the application:**
    - Place the project in your web server’s root (e.g., `htdocs` for XAMPP).
    - Visit `http://localhost/aurora-theater` in your browser.
